Odev programın veri okuyabilmesi için, klasör hiyerarşisinden ./data yoluna gidilmelidir.
veri dosyasının ismi "veriler", uzantısı ise ".txt" olmalıdır.

örneğin:
	./data/veriler.txt

The project only accepts data from ./data path as ".txt" format.
The data file must be named as "veriler".
example apply:
	./data/veriler.txt
